Como obter a conquista Pull Shark

Para obter o selo Pull Shark, você precisa fazer 2 pull requests mesclados. Você receberá então o selo Pull Shark

  1. Você precisa fazer um fork de um repositório (você pode fazer um fork de qualquer repositório que desejar).
  2. Quando seu repositório bifurcado for criado, você precisará fazer algumas alterações nele (como adicionar um novo arquivo ou modificar o código!).
    Depois disso, você precisará acessar a aba de pull requests;
  3. Clique no botão "Criar solicitação de pull";
  4. Agora você pode ver o texto verde indicando que é possível mesclar sua solicitação de pull request. Portanto, clique no botão "Criar solicitação de pull request"
    para criar sua solicitação com sucesso. Em seguida, você precisa aguardar até que sua solicitação seja mesclada pelo criador do repositório.
  5. Pronto! Agora você pode ver a conquista "Puxar Tubarão" na sua lista de conquistas.
